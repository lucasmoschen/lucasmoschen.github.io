In this project, I could understand problems with an optimization process,
when one can control a system with some variable. For example, in the
Coronavirus Pandemic, we have a system that describes the infected,
susceptible and recovered, for example. However, the governments can interfere
this system establishing a *lockdown* or through a vaccine. These are examples
of controls. So this topic has a lot of illustrations and applications in
biology and for that reason, we chose the reference book to be "Optimal
Control Applied to Biological Models", written by Suzanne Lenhart and John T.
Workman. 

Notes based on this book were produced in my native language to serve as a start point
reference in my language and notebooks with the applications also, all
developed in *Python* programming language. 

[Download the book
here](https://github.com/lucasmoschen/Optimal_Control_Biological/blob/master/notes/book.pdf).
